**Zobly is a full-stack job portal platform that connects job seekers with employers, allowing companies to post jobs and manage applicants while enabling candidates to apply seamlessly. The platform focuses on clean UX, secure authentication, and real-world job portal functionality.

###  Tech Stack
- **Frontend:** React.js Tailwind CSS
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL (via Prisma ORM)  

###  Features
## Job Seekers
- Register & login as jobseeker
- Browse and search all available jobs
- View detailed job descriptions
- Apply for jobs with resume upload
- Track applied jobs (My Applications)

## Employers
- Register & login as employer
- Post new job listings
- Edit & delete existing jobs
- View applicants for posted jobs
- Access full applicant details & resumes
